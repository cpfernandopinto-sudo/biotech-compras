# assets/

Recursos estáticos do sistema Biotech Compras.

## Estrutura esperada

```
assets/
├── logo/
│   ├── biotech-logo.svg
│   ├── biotech-logo-dark.svg
│   └── biotech-logo-white.svg
├── icons/
│   └── favicon.ico
└── images/
    └── og-image.png
```

## Diretrizes de uso

- Logos em formato `.svg` (vetorial, escalável)
- Ícones em `.svg` ou `.png` (mínimo 64×64px)
- Imagens otimizadas para web (máx. 200kb cada)
